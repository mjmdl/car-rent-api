# Car Rent API

A car renting API built with [Nest](https://github.com/nestjs/nest) framework and TypeScript.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
